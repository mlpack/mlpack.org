Downloading and Installing mlpack

Prebuilt mlpack packages are available for most operating systems; these packages may not be the latest version, so if you are encountering issues, use the official source release.

Source download and build reference

  • Latest stable source release: mlpack-4.5.0.tar.gz; released May 28, 2024
  • README build directions
  • C++ quickstart installation guide
  • Windows build guide

C++ system-wide package installation

  • Debian/Ubuntu:
    sudo apt-get install libmlpack-dev
  • Red Hat/Fedora:
    sudo dnf install mlpack-devel
  • Arch Linux:
    sudo pacman -S mlpack
  • macOS (Homebrew):
    brew install mlpack
  • macOS (MacPorts):
    sudo port install mlpack
  • Windows (vcpkg):
    vcpkg install mlpack
  • Windows MSI installer:
    mlpack-4.5.0.msi

Language-specific installation

  • Python (pip):
    pip install mlpack
  • Python (conda):
    conda install -c conda-forge mlpack
  • Julia:
    import Pkg; Pkg.add("mlpack");
  • R:
    install.packages("mlpack")
  • Go:
    go get -u -d mlpack.org/v1/mlpack
