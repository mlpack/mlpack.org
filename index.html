mlpack
fast, header-only C++ machine learning library
with bindings to Python, Julia, R, Go, and the command-line
  • Written in C++ and built on Armadillo, ensmallen, and cereal.
  • Aims to provide fast, extensible implementations of cutting-edge machine learning algorithms.
  • Read the release announcement.

Latest version: 4.4.0  •  6.0MB •  released May 28, 2024
